# README del Proyecto

- 📁 [Archivo](./Proyecto%201/Proyecto1.pkt)`.pkt` – Archivo de simulación o configuración del proyecto.
- 💰 [Presupuesto](./Proyecto%201/Presupuesto.md) – Archivo detallando los costos y presupuesto asociados al proyecto.
